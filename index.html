電気分解の基礎マスター

電気分解は、外部から電気エネルギーを与えることで、自発的には起こらない酸化還元反応を強制的に起こす操作です。
どの電極で何が発生するかは、「優先順位のルール」を上から順番に確認するだけで簡単に判別できます。

1. 陰極（－極）の反応ルール

陰極では、電子（e-）を受け取る還元反応が起こります。
水溶液中の陽イオン（＋イオン）に注目し、以下の順番でチェックします。

🔵 陰極のチェックフロー

  1. イオン化傾向の小さい金属イオン（Ag+, Cu2+など）はあるか？
    → YES：その金属が単体として析出する。
    Cu2+ + 2e- → Cu
  2. （上記がない場合）水溶液か？
    → H+ または 水（H2O）が電子を受け取り、水素（H2）が発生する。
    2H2O + 2e- → H2 + 2OH-

※Na+やK+などのイオン化傾向が大きい金属イオンは、水溶液中では析出しません（水の方が先に還元されるため）。

2. 陽極（＋極）の反応ルール

陽極では、電子（e-）を奪われる酸化反応が起こります。
ここでは「電極の種類」と「陰イオン（－イオン）」に注目します。

🔴 陽極のチェックフロー

  1. 電極が「Pt（白金）」「C（炭素）」以外の金属（Cu, Agなど）か？
    → YES：電極そのものが溶ける。
    Cu → Cu2+ + 2e-
  2. （電極がPtかCの場合）ハロゲン化物イオン（Cl-, I-など）はあるか？
    → YES：ハロゲンの単体（Cl2, I2など）が発生する。
    2Cl- → Cl2 + 2e-
  3. （上記がない場合）水溶液か？
    → OH- または 水（H2O）が電子を失い、酸素（O2）が発生する。
    2H2O → O2 + 4H+ + 4e-

※SO42- や NO3- のような多原子イオンは、非常に酸化されにくいため変化しません。
